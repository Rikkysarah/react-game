# Project Title
Game App with React.js

## INSTRUCTIONS
1. Clone the repository
  ```
  $> git clone https://github.com/Rikkysarah/react-game.git
  ```
2. Install all the dependencies. Make sure you are inside the `game` folder.
  ```
  $> npm install
  ```

3. Start the server.
  ```
  $> npm start
  ```
## OVERVIEW
 An interactive tic-tac-toe game.
